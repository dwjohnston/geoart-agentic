== Project Overview ==

This is a generative art engine called "Geometric Art Engine" (geoart-agentic).
Graphs of connected nodes are evaluated each frame to produce animations drawn to a canvas.

== Tooling ==

- Runtime: Bun (used for node-like commands)
- Test command: bun test <fileName>
- Type checking: bun typecheck
- Full validation: bun validate (runs all tests and static checks)
- Code generation: bun generate (runs code generation scripts)

== Architecture ==

The project uses a node-based graph architecture. Node types include:
- Compute nodes (src/nodes/compute)
- Render nodes (src/nodes/render)
- Control nodes (src/nodes/control)
- Module nodes (src/nodes/module)
- Schema definitions (src/schema)
- Algorithms (src/algorithms)
